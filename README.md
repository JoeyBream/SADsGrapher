# SADsGrapher

Interactive treemap dashboard visualising the global farmed animal suffering burden using **Suffering Adjusted Days (SADs)** data by Sam Hilton.

Rectangle area = Total SAD Burden (SADs per animal x animals per year). Concept of SADs from [Ambitious Impact (AIM)](https://www.charityentrepreneurship.com/post/sads-suffering-adjusted-days-an-improved-animal-welfare-measurement-tool).

Vibe coded with Claude Code

## Run locally

```
npm install
npm run dev
```

## Build

```
npm run build
```
