# SADsGrapher

Interactive treemap dashboard visualizing the global farmed animal suffering burden using **Suffering Adjusted Days (SADs)** data.

Rectangle area = Total SAD Burden (SADs per animal x animals per year). Data from [Ambitious Impact (AIM)](https://www.charityentrepreneurship.com/post/sads-suffering-adjusted-days-an-improved-animal-welfare-measurement-tool).

## Run locally

```
npm install
npm run dev
```

## Build

```
npm run build
```
